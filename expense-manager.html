Installation Android

Install the dependencies and devDependencies and start the metro server. 

cd <Project_folder>
npm i
npx react-native link
npm run android

Note: Sometime gradle does not sync. So open the project folder in android studio to fix the issue

Installation IOS

Install the dependencies and devDependencies and start the metro server. 

cd <Project_folder>
npm i
npx react-native link
cd ios && pod-install
npm run ios

Note: Use Xcode 12 & above
